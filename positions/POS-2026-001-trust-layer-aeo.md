WORLD-2026-001 の本線「信用レイヤー収斂」に張るポジション。計測ツール（つるはし）ではなく、AI が安心して参照できる一次情報の構造化・検証・供給側に立つ。

# 1. エンジニアリング — ブレークスルーか、段階的改善か

変数化を狙う定数：**「専門家の一次情報は構造化されない」**（LAW-002 の帰結として放置されてきた誘因の定数）。AI 引用の計測自体は段階的改善（つるはし）であり、それ単体では LAW-006 の寿命に従う。ブレークスルー条件は、計測データを使って「発信しない専門家の知識を、本人の発信コストゼロで引用可能な形式に変換する」パイプラインの実証。SIG-2026-002（引用は情報の有無でなく形式で決まる）が、変換だけで引用率を桁で動かせる可能性を示す S0 証拠。

# 2. タイミング — なぜ今か

関所の交代期（HIST-001 第7行）だけに開く窓。PageRank 期に SEO 産業が生まれた位置が、いま空いている。窓の定義：**D1（Google/OpenAI の公式可視性ダッシュボード）が観測される前**が参入窓であり、観測されたら計測側の窓は閉じる（STATE-content の最重要ウォッチ）。早すぎ判定：AI 検索経由の商取引比率が計測不能なほど小さいままなら早すぎ。現状 SIG-2026-001（ゼロクリック化の進行）は窓が開いていることを示す。

# 3. 独占 — 大きなシェアを取れる小さな市場から始めているか

「AEO ツール全般」は混雑するつるはし市場（LAW-006）であり独占不能。独占可能な最小市場は**業種特化の引用データ資産**：特定業種（例：日本の特定専門サービス業）について、AI が何を訊かれ・何を引用するかの S0 観測データを持つのは自社のみ（GPR 実験）。ボトルネック命中（LAW-007）の確認：この業種の顧客のボトルネックは「コンテンツ量」ではなく「引用される形式と信用の担保」——生成コストは解凍済み（CON-003）なので量産ツールはボトルネック非命中で外れる側。

# 4. 人材 — 正しいチームか

要求される S0 アクセス：(1) GPR 実験基盤（保有済み）、(2) 業種の沈黙する専門家への接触経路（tkbase の顧客基盤）。CASE-2015 の教訓を適用すると、現状の弱点は「見抜く側」は揃っているが「殴り切る資本」が未確保であること。反世界分岐（p=0.30）が実現した場合に備え、資本集約的な計測インフラ競争には入らない編成を維持する。

# 5. 販売 — プロダクトを作るだけでなく、届ける方法があるか

LAW-001 の第4変換（意思決定接続）で答える：計測レポート（現状把握）→ 改善実装（引用される形式への変換）→ 成果の再計測、という閉ループを顧客の既存マーケ予算（SEO 予算の付け替え）に接続する。「AI に載らなければ」という言説の流通（LAW-003 の期待）が販売の追い風だが、期待で売ると期待の剥落と共に解約される——**成果（引用の実測変化）で売る**ことが LAW-003 の焼死体側に入らない条件。

# 6. 永続性 — 10年後も生き残るポジションか

錨を下ろす不変量：CON-002（信用は責任を要求する — AI が生成できないもの）と CON-001（引用枠の希少性 — 注意の関所は位置を変えて存続）。この2つが frozen である限り、「誰の情報を信じるか」の判定需要は消えない。反世界（プラットフォーム吸収、p=0.30）が実現した場合の生存形：計測機能は死ぬが、**業種特化の構造化一次情報データ資産**はプラットフォームにとっても調達対象になる（PRED-2026-000 の出版社ライセンスの業種版）。つるはしではなく Levi's 型（データ資産・信用ブランドへの転換）を最初から設計に入れる。

# 7. 隠れた真実 — 他社が気づいていない何を知っているか

コンセンサスの地図：「AEO は SEO の延長であり、コンテンツを増やして計測すれば勝てる」。
自分の領土観測とのズレ：(1) SIG-2026-002 が示すのは、引用は量ではなく**形式と一次性**で決まる——つまり勝負は生成側でなく構造化側にある。(2) LAW-002 により、最も引用価値の高い情報はまだ Web に存在しない（沈黙する専門家の中にある）。全員が「既にある情報の順位争い」をしている間に、**まだ発信されていない一次情報の供給側**を押さえるのが本当のゲーム。この秘密は S0（自社実験）と採点済み法則（LAW-005 系の PRED-2026-000 的中、Brier 0.023）で裏づけられている。

# 総合判定

最も弱い答えは **4（人材/資本）** と **2（タイミングの窓の閉じ方）**。次のアクション：(1) D1 の監視頻度を上げる（STATE-content の更新周期 30日は D1 に対しては粗い）、(2) PRED-2026-002（専門家対価プログラム）の解決が「供給側ゲーム」の妥当性を検証する最初の審判——2028-06 を待たず、四半期ごとに先行指標を state に記録する。
