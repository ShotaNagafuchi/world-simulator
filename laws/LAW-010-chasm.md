# 主張

技術普及はS字カーブを描くが、カーブは連続ではない。アーリーアダプター（市場の約16%）とアーリーマジョリティの間に構造的な断絶（キャズム）がある（Moore 1991）。この断絶の原因は、2つの層の評価基準が**逆転**するからである：

- **アーリーアダプター**: 「新しいから使う」「可能性に賭ける」「不完全でも先行者利益がある」→ ビジョン駆動
- **アーリーマジョリティ**: 「みんなが使っているから使う」「実績がないと信用しない」「リスクを避けたい」→ 証拠駆動

アーリーアダプターに響くメッセージ（革新性、可能性、先進性）は、マジョリティには**逆効果**（不安定、未検証、リスク）として作用する。したがってアーリーアダプターでの急成長は、マジョリティでの成功を予測しない。

# メカニズム

キャズムの根底にある制約の組み合わせ：

1. **CON-005（社会的証明）**: マジョリティは「他の人が使っている」を要求する。しかしアーリーアダプターの「使っている」はマジョリティの参照集団に入らない（「あの変わった人たちが使っている」は社会的証明にならない）
2. **CON-010（損失回避）**: マジョリティは「失敗したくない」が優先。利得より損失回避が強い
3. **CON-011（現状維持バイアス）**: マジョリティはデフォルトから動かない。アーリーアダプターはデフォルトを変えることに快楽を感じる（CON-007の地位欲求——先進的であることは地位シグナル）
4. **CON-012（認知的一貫性）**: マジョリティは「自分は慎重な人間」というアイデンティティと一致する選択をする。新しいものに飛びつくことはアイデンティティと矛盾する

キャズムを越えるために必要なもの：
- アーリーアダプターの言語からマジョリティの言語への翻訳（ビジョン → 実績・事例・リスク低減）
- マジョリティの参照集団の中に導入事例を作る（「同じ業種の同規模の会社が使っている」）
- CON-006（10倍便利）がキャズムを力技で超える唯一の方法——利得が損失回避の閾値を大幅に超えると、評価基準の違いが無関係になる

# 証拠

- CASE-2008-tesla: Roadster（アーリーアダプター向け、高性能スポーツカー）→ Model S（アーリーマジョリティ向け、高級セダン）→ Model 3（マジョリティ向け、大衆価格）の3段階で、各段階で**ナラティブ、価格、機能、ターゲット**をすべて変えた。Roadster のままでは Mass market に到達しなかった
- AI市場（現在進行中）: ChatGPTはアーリーアダプターで急速に普及したが、エンタープライズ採用は「セキュリティ」「コンプライアンス」「実績」を要求する——評価基準の逆転がリアルタイムで観測されている

# 反証可能性

新しい技術製品の普及曲線を記録し、16%前後でS字カーブに断絶があるか追う。断絶なく連続的に普及する製品が主流になったらこの法則は死ぬ。

# 予測への使い方

入力：(1) 技術の現在の普及率、(2) 現在のユーザー層（アーリーアダプターかマジョリティか）、(3) 現在のメッセージング・価格設定
出力：(1) キャズム手前か後か、(2) キャズム手前なら、マジョリティの評価基準への変換が行われているか、(3) CON-006 的な10倍利得があるなら力技で超えられるか

**LAW-003（期待資本）との結合**: キャズム手前の急成長は期待資本を引き寄せる。しかしキャズムでの停滞は期待の崩壊を引き起こし、資金が引く。「成長が止まった → ナラティブ崩壊 → 資金引き上げ → 本当に成長が止まる」のスパイラル。
