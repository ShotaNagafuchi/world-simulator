# 主張

「偏りのない評価」を提供するレイヤー（格付け・比較・推薦・検索）の中立性は、収益源の位置で決まる。収益が評価される側（売り手）から出るなら、中立性は個人の倫理と無関係に、時間の問題で構造的に崩壊する。しかも崩壊後も事業は儲かり続けるため、市場の自浄は働かない。

課金主体は3通りしかなく、それぞれ固有の崩壊モードを持つ：
- **買い手課金**：中立を保てるが、評価結果は複製可能な公共財なので回収が崩れやすい（Consumer Reports 問題）
- **売り手課金**：回収は容易だが中立性が腐る（格付け機関、比較サイト、食べログ型）
- **第三者課金（広告・規制）**：中立性は第三者の目的に従属する（広告なら注意の切り売り、規制なら政治）

# メカニズム

審判の売上が被評価者の満足に比例する構造では、厳格さは売上の敵になる。競合審判が存在すると被評価者は甘い審判を選べる（レーティング・ショッピング）ため、審判間の競争はむしろ寛大さの競争になる——市場競争が品質を上げるという通常の力学が、審判業では逆向きに働く。これは腐敗が「起きるかも」ではなく均衡である理由。

延命する設計は3つ：(a) 計測と推奨の分離（診断料は取るが順位は売らない監査法人型の分離壁）、(b) 買い手課金＋複製防止（データが腐る速さで守る：評価が動的に更新され続けるなら公共財化しにくい）、(c) 規制による義務化（監査法人型。ただし寡占とレント化が代償）。

# 証拠

- CASE-2008-rating-agencies: 主証拠。買い手課金の崩壊 → issuer-pays → レーティング・ショッピング → AAA インフレ → 金融危機。
- CASE-1998-google-pagerank: organic/ads 分離壁は当初機能したが、広告表示の同化と自社サービス優遇（各国の独禁法訴訟）で漸進的に浸食された。分離壁は腐敗を止めず、遅らせる。

# 反証可能性

「偏りのなさ」を主張する評価レイヤーの収益構成（売り手課金比率）と、独立監査・研究による偏り検出結果を対応させて記録する。売り手課金比率が高いまま長期中立を保つ主要事例が積み上がれば、この法則は死ぬ。

# 予測への使い方

入力：評価・比較・推薦レイヤーの収益源の位置。
出力：(1) 売り手課金の「中立」レイヤーは中立性の腐敗を予測する（時期は分離壁の設計品質に依存）、(2) 自分が中立レイヤーを建てる場合の設計審査——**最大の敵は競合ではなく自分の収益モデル**、(3) 既存審判の腐敗は、買い手課金または監査分離型の新規参入の窓が開いたことを意味する。
