# 主張

すでに信用があり、発信しなくても仕事が来て、時間単価が高い人ほど、発信の限界効用が低い。したがって「発信支援市場」の顧客は情報価値の低い層に偏り、最も価値ある一次情報は市場の外に留まる。発信パイプラインの真の価値は「発信したい人の支援」ではなく「発信すべきだが発信しない人への誘因設計」で決まる。

# メカニズム

発信の期待リターン = （新規信用の獲得）−（発信コスト＋炎上・責任リスク）。既に信用が飽和している者にとって新規獲得分はほぼゼロで、リスクだけが残る。逆に信用がない者は発信の期待リターンが大きい。結果、発信量と情報の質は構造的に逆選択される。これは市場の一時的な歪みではなく、インセンティブの定常状態である。

# 証拠

- CASE-1849-gold-rush: 本当に儲かる鉱脈情報を持つ者はそれを公表せず自分で掘るか売り抜けた。公表された「金が出る」という情報は、既に価値が薄まったものだった。

# 反証可能性

高信用専門家の発信率を定点観測する（例：士業・医師の SNS/発信プラットフォーム参加率）。誘因設計（対価・匿名性・代理発信）なしで参加率が跳ねたら、この法則は死ぬ。

# 予測への使い方

入力：ある領域の一次情報保有者の信用飽和度と時間単価。
出力：その領域で機能する誘因設計の型（対価直接支払い / 代理発信・ゴーストライティング / 発信の副産物化＝業務データの自動構造化）。「本人に発信させる」型のサービスは高価値層には届かない、という予測。
