# 主張

供給の爆発は必ずフィルタへの需要を生む。あるレイヤーの生産コストがゼロに近づくと、価値は「作ること」から「何を信じてよいかを判定すること」へ移動する。AI によるまとめ・生成が容易になるほど、「どの発信源が本当に価値を持つか」を判定するレイヤーが情報流通の最上流の関所になる。

これは LAW-004（摩擦低下）の系である：AI が下げない摩擦——信用、実在性、責任——が残余ボトルネックとなり、そこに価値が再集中する。

# メカニズム

情報の消費者（人間・AI・企業）の注意と検証能力は有限である。供給が10倍になっても消費は10倍にならないため、選別の経済価値が供給量にほぼ比例して増える。さらに生成コストの低下は偽情報・低品質情報のコストも下げるため、信用の判定はより困難かつ高価になる。困難かつ需要が増えるものに価値が集中する。

歴史的な反復：活版印刷 → 出版社と検閲、Web → Google、金融商品の複雑化 → 格付け機関、企業の大規模化 → 監査法人。

# 証拠

- CASE-1998-google-pagerank: Web ページ生成コストの低下（誰でも作れる）→ リンク構造を「信用のシグナル」として評価する PageRank が情報流通の関所となり、コンテンツ生産者全体より多くの利益を取った。
- CASE-1849-gold-rush: ラッシュ終息後に生き残ったのは、つるはし屋ではなく信用ブランドへ転換した者（Levi's、Wells Fargo）だった。

# 反証可能性

「AI 可視性・引用計測」「出所証明（C2PA 等）」「AI 向け情報検証」カテゴリへの投資額・売上を毎年記録し、生成系カテゴリと比較する。5年間、相対縮小が続いたら死ぬ。

# 予測への使い方

入力：あるドメインでの供給爆発の兆候（生成コスト低下率、コンテンツ流通量）。
出力：そのドメインで次に立ち上がる信用レイヤーの形態（誰の信用を、誰に対して、何のフォーマットで判定するか）と、そこへの資金流入時期の予測。
