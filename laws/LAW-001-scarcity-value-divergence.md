# 主張

希少な一次情報を持つことと、それが市場価値になることは別である。情報は ①発信される ②構造化される ③信用が付与される ④意思決定に接続される、の4変換を通過した割合（変換率）に比例してのみ価値化される。変換率がゼロなら、どれほど希少でも価値はゼロ。

# メカニズム

買い手（企業・AI・意思決定者）が支払うのは「情報」ではなく「意思決定の改善」である。生の情報は検索できず、検証できず、責任の所在がなく、意思決定プロセスに入らない。だから価値の大半は変換レイヤーに落ちる。過去に「デジタル化すべき情報」が大量に存在しながらデジタル化されなかったのは、希少性が足りなかったからではなく、変換のコストと誘因が釣り合わなかったからである。

# 証拠

- CASE-1998-google-pagerank: Web上のコンテンツ（情報そのもの）の生産者より、それを構造化・信用評価・意思決定接続した Google が価値の大半を取った。

# 反証可能性

kill condition の観測方法：データマーケットプレイス（生データ直販）の市場規模と、変換レイヤー（検索・分析・信用評価）の市場規模の比率を追う。前者が後者を上回る状態が2年続いたらこの法則は死ぬ。

# 予測への使い方

入力：ある領域の一次情報の希少性と、現在の変換率（発信率・構造化率・信用付与率・接続率）。
出力：価値が集中する地点は「変換率が最も低いボトルネック工程」である、という予測。
