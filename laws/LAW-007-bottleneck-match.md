# 主張

LAW-004（摩擦低下による資金流動の再配線）の必要条件を精密化する。技術がある摩擦を桁で下げても、**その摩擦が当該産業のボトルネック制約でなければ、産業は変わらない。** 変化の予測は「何の摩擦が下がるか」だけでは足りず、「その産業のボトルネックはどの層（物理／本性／制度／慣行／技術コスト）の制約か」の宣言とセットでなければならない。

# メカニズム

産業のスループットは最も強い制約（ボトルネック）で決まる（制約理論）。ボトルネック以外の工程を100倍速くしても全体は1ミリも速くならず、資金は動く理由を持たない。インターネットが「すべてをデジタル化する」と予測された時、変化の駆動力（伝達コストの崩壊）は正しく特定されたが、産業ごとにボトルネックが伝達コストなのか、信用・責任・誘因・物流なのかの分類が抜けていた。結果、予測は産業ごとにバラバラに当たり、外れた。

系：**予測の誤りは2種類に分類できる。** (a) 楽観の誤り＝遅い層（制度・本性）の制約を速い層（技術）と誤認する。(b) 保守の誤り＝速い層の制約を定数と誤認する（→ CASE-2015-spacex-reusability）。

# 証拠

- CASE-2000-undigitized-industries: 同じ「伝達コスト崩壊」を受けて、音楽は完全デジタル化し、不動産・医療は30年変わらなかった。差はボトルネックの層の違いで完全に説明できる。
- CASE-2015-spacex-reusability: 打ち上げコストは40年間「定数」とされたが、ボトルネックの正体が物理ではなく制度（cost-plus 契約のレント）だと見抜いた者が変数化した。

# 反証可能性

新しい摩擦低下イベントごとに「ボトルネック一致か否か」を事前に判定してから資金流動の予測を出し、的中率を非一致ケースと比較する。一致判定に予測力がなければこの法則は死ぬ。

# 予測への使い方

入力：対象産業のボトルネック制約とその層（constraints/ を参照）、下がりつつある摩擦のリスト。
出力：(1) 摩擦低下がボトルネックに命中する産業＝変わる、(2) 命中しない産業＝「変わる」という言説だけが流通して期待資本が焼かれる（LAW-003 と結合すると「外れる側のゴールドラッシュ」を事前識別できる）。
