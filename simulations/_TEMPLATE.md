# 前提となる法則の合成

どの法則をどう組み合わせてこの世界線群を導出したか。

# 分岐

各分岐について：
- 記述（2030年に何がどうなっているか）
- 確率
- 先行指標（どのシグナルが観測されたらこの分岐の確率を上げるか）
- この分岐で勝つポジション / 死ぬポジション

# 決定的観測（Value of Information）— 必須

答えが分岐確率を**最も動かす**観測から順にランクづけする。D1, D2, ... と番号を振る。

| # | 観測 | 観測された場合の確率移動 | どこで観測するか |
|---|------|------------------------|-----------------|
| D1 | 最も決定的な観測 | 例: 反世界 +0.15 / 本線 -0.15 | 出典・URL・データソース |

- 「答えを知っても確率が動かない観測」はリストに入れない。網羅は禁止
- state/ の定点更新はこのリストを上から巡回する

# 掟

- 反世界（anti-thesis）の分岐は必須。確率 0.2 未満にする場合は理由を明記
- 分岐の確率の合計は 1.0。「未知」に必ず残余を割り当てる
- 各分岐は predictions/ の予測とリンクし、予測の解決が分岐確率の更新トリガーになる
- 分岐確率を更新したら、ファイル末尾の「更新履歴」に日付・変更・根拠となった観測を追記する
