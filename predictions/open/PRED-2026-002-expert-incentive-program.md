# 賭けの内容

LAW-002 が正しければ、最も価値ある一次情報の保有者は自発的には発信せず、AI企業は Web 上の公開情報を汲み尽くした後、必ず「発信されていない知識」に対価を払う段階に進む。出版社ライセンス（PRED-2026-000、resolved true）は組織への支払いだった。次の段階は個人の専門家への誘因設計である、という賭け。

確率を0.45に抑えた理由：支払いの管理コスト（真贋検証・品質評価・契約）が高く、AI企業は個人ではなく「専門家を束ねるアグリゲーター」経由を選ぶ可能性が高いため。その場合この statement は false になるが、LAW-002 自体は「アグリゲーター＝誘因設計レイヤーの成立」として別の形で支持される。

# この予測が外れるとしたら

上記の通りアグリゲーター経由になる形。false になった場合、失点は statement の設計（直接/間接の区別）に帰属し、LAW-002 の反証とはしない——この切り分けを resolution_note に必ず書くこと。
