# 賭けの内容

確率 0.3 ——つまり「引き受けない」側に賭ける。根拠は CON-002（信用は責任の所在を要求する）の裏返し：網羅を宣言することは責任の引き受けであり、プラットフォームは規模ゆえに責任回避の誘因を持つ（全カテゴリ×全地域で母集団を保証するコストと訴訟リスクは巨大）。さらに LAW-008 により、プラットフォームの推薦は広告収益と構造的に融合していく方向にあり、「中立な網羅」を自己宣言する動機は薄い。

この予測は POS-2026-002（網羅の監査レイヤー）の窓を測るリトマス試験紙である：
- **false（引き受けない）なら**：網羅性の保証は独立レイヤーに構造的に残る。ポジションの空間が確認される
- **true（引き受ける）なら**：ナイーブな「よりよいAI検索」ポジションは死ぬ。ただし自己宣言された網羅を**外部から検証する**監査需要はむしろ発生する（LAW-008：審判は自己監査できない）

# この予測が外れるとしたら

最も可能性の高い外れ方：規制（EU等）が推薦の透明性・母集団開示を義務化し、プラットフォームが「自発的にではなく強制されて」引き受けるケース。この場合 statement は true になるが、原因は責任回避誘因の消失ではなく規制世界分岐（WORLD-2026-001）の実現であり、resolution_note で必ず切り分けること。
