# 何が起きたか

2006年公開の「マスタープラン」は、高級スポーツカー → 高級セダン → 大衆車という段階戦略を事前に宣言した。量産実績ゼロ、業界の常識では新規参入不可能（米国で最後に成功した自動車新興は1920年代）とされる中、Tesla はこのナラティブで資金調達を繰り返し、2008年の倒産寸前を生き延び、2020年に時価総額でトヨタを抜いた。

# どの摩擦が下がったか

リチウムイオン電池のコスト：2010年 約$1,100/kWh → 2020年 約$140/kWh（約1/8）。EV の成立可能性は思想でも規制でもなく、ほぼこの一本のコスト曲線の関数だった。Tesla の本質的な賭けは「電池コスト曲線が降り続ける」ことへのレバレッジド・ロングである。

# 資金はどこからどこへ流れたか

公開市場・政府融資（DOE 4.65億ドル）・戦略投資家から、量産能力（ギガファクトリー）へ。重要なのは順序：**成果の実証より先に、ナラティブが資金を調達し、その資金が成果を作った。** 期待→資本→実体の順であり、実体→資本ではない。既存メーカーは「EV に需要はない」と実績データで判断し、コスト曲線という先行指標を読み損ねた。

# 法則への接続

- LAW-003: 期待資本の設計の教科書。マスタープランの機能は事業計画ではなく「単純で検証不能だが追跡可能なナラティブ」の公開装置だった。
- LAW-004: EV 普及の予測は「消費者の環境意識」からは導けず、$/kWh 曲線からはほぼ機械的に導けた。摩擦のコスト系列こそが予測変数。

# この事例が法則に反する側面（必須）

- 同じ期待資本を調達した EV 新興（Fisker、Better Place、後の Nikola 等）はほぼ全滅した。LAW-003 は資金が**流れ込む**ことを予測するが、受け手の生存は予測しない。期待資本は生存の必要条件であって十分条件ではない。
- Tesla は「つるはし」ではなく「掘る側」で勝った——LAW-006 の反例に見える。ただし充電網・ソフトウェア・電池内製で自らインフラ側に転換した点は LAW-005（信用・インフラへの転換が生存条件）と整合する。
