# 何が起きたか

1990年代後半、「情報はすべてデジタルに載り、リアルは体験、所有・取引はデジタルへ」という予測が支配的だった。25年後の結果は産業ごとに割れた：音楽・ニュース・地図・写真は完全デジタル化。小売は部分的（EC化率は主要国で2-3割程度で頭打ち傾向）。不動産取引・医療記録・行政手続は、日本では2020年代までFAX・対面・紙が残存した（重要事項説明のオンライン化解禁は2021年、コロナ禍という外生ショックが必要だった）。

これは「起きたこと」ではなく「**起きなかったこと**」を記録する反実仮想の事例である。予測の外れ方こそが法則の原料になる。

# どの摩擦が下がったか

下がったのは情報の伝達・複製コストのみ。**産業ごとに、それがボトルネックだったかどうかが運命を分けた：**

| 産業 | ボトルネックの正体 | 層 | 結果 |
|------|-------------------|-----|------|
| 音楽 | 複製・流通コスト | tech-cost | 完全デジタル化（ただし価値は関所=Spotify/レーベルへ） |
| 小売 | 物流（モノは圧縮できない） | physics | Amazonは「デジタル企業」ではなく倉庫と物流の会社になることで勝った |
| 不動産取引 | 信用・責任・免許制度 | institution / human-nature | 情報掲載はデジタル化（ポータル）、取引自体は変わらず。価値はポータル（広告関所）だけが取った |
| 医療記録 | 責任・規制・デジタル化の誘因欠如（入力コストは医師が払い、便益は他者が得る） | institution + 誘因 | 30年進まず。LAW-002 の誘因逆転の教科書 |
| 行政 | 制度そのもの | institution | 外生ショック（コロナ）まで不動 |

# 資金はどこからどこへ流れたか

期待資本は「すべての産業がデジタル化する」前提で全方位に流れ（dot-comバブル、LAW-003）、ボトルネック一致産業でのみ回収された。非一致産業では、資金は産業を変えられずに**情報掲載レイヤー（ポータル・広告）だけを太らせた**——産業のコア取引に入れず、周縁の広告関所に堆積した。

# 法則への接続

- LAW-007: この事例が主証拠。同一の摩擦低下が産業ごとに正反対の結果を生み、差はボトルネックの層で完全に説明できる。
- LAW-004: 摩擦低下の再配線は起きたが、「命中した産業でのみ」という条件つきだった。
- LAW-002: 医療記録の停滞は誘因の逆転（コスト負担者と便益享受者の分離）で説明される。デジタル化の失敗の多くは技術ではなく誘因設計の失敗。

# この事例が法則に反する側面（必須）

- 行政のデジタル化はコロナという外生ショックで一気に動いた。つまり institution 層の制約は「ゆっくり解凍する」のではなく、**危機で不連続に跳ぶ**。層の速度モデル（遅い層=ゆっくり）は平時にしか成り立たず、ショック時の非線形性を過小評価する。
- エストニアや中国のモバイル決済のように、レガシー制度が薄い場所では institution 層が最速で動いた（跳躍）。層の速度は絶対値ではなく、**既存制度の厚みとの相対値**である。同じ産業でも国によってボトルネックの層が違う——予測は産業単位でなく産業×制度環境単位で行うべき。
