# 何が起きたか

1848年の金発見後、約30万人がカリフォルニアに殺到した。金鉱夫の大半は装備・食料・宿泊のコストで費用倒れになり、平均的な鉱夫のリターンは賃金労働を下回った時期が長い。一方、卵1個が現在価値で数十ドル、シャベルや宿泊が本土の数十倍で売れた。サンフランシスコの人口は約2年で1,000人規模から2.5万人規模へ膨張した。

# どの摩擦が下がったか

金の発見そのものは摩擦低下ではなく「期待の点火」である。むしろ摩擦（輸送・装備・情報・金融）は高いままであり、**高いままの摩擦を保有していた者に資金が流れた**。Samuel Brannan は金発見の情報をいち早く掴み、自分では掘らず、街中で「金が出たぞ」と宣伝して事前に買い占めたつるはし・シャベルを売った——期待の増幅と道具の販売を一人で垂直統合した。

# 資金はどこからどこへ流れたか

東部・海外の貯蓄と労働力 →（期待を経由して）→ 輸送業者、装備商、宿泊、酒場、金融・郵送（Wells Fargo）、報道。掘る側の総支出が金の総産出額を上回る局面が存在した。つまりこの市場の実体は「金の市場」ではなく「金への期待の市場」だった。

# 法則への接続

- LAW-003: 資金は産出実績ではなく期待に流れた。ナラティブ（「拾うだけで金持ちに」）の単純さが流入量を決めた。
- LAW-006: 道具・輸送・金融・情報のレイヤーは、鉱夫の成否と無関係に収益を確定させた。
- LAW-002: 本当に儲かる鉱脈情報を持つ者はそれを公表しなかった。公表された情報は価値が薄まったものだった。
- LAW-005: ラッシュ終息（1855頃）を生き延びたのは、無名のつるはし商ではなく、期間中の顧客接点を**信用ブランド**（Levi's）と**信用インフラ**（Wells Fargo の輸送・銀行網）に転換した者だけだった。

# この事例が法則に反する側面（必須）

- 掘る側にも大勝者は実在した（初期参入者の一部は巨富を得た）。「つるはしが常に勝つ」のではなく、勝者の**分散**が違うだけである。LAW-006 を「参加者は儲からない」と読むのは誤読。
- つるはし商の大半もまた消滅した。生存はレイヤー選択ではなく信用転換の有無で決まった——つまり LAW-006 単独では生存を予測できず、LAW-005 との結合が必要。
