# 何が起きたか

1999年、Confinityがオンライン決済サービスを開始。2000年にX.comと合併しPayPalに。eBayの出品者間決済に浸透し、2002年のIPO時点で登録ユーザー数1,600万人。同年eBayが15億ドルで買収。2004年までにeBay取引の70%以上がPayPal経由。既存銀行もオンライン決済を試みたが（Citibank c2it, BankOne eMoneyMail等）、すべて撤退した。

# どの摩擦が下がったか

インターネットが「見知らぬ個人間の少額取引」の摩擦を桁で下げた。従来はこの取引自体が不可能か、小切手郵送（数日＋手数料）だった。PayPalはメールアドレスだけで即時送金を可能にし、取引コストを実質ゼロに近づけた。

# 資金はどこからどこへ流れたか

**Before**: 個人間の遠隔取引はほぼ存在しない。銀行間送金（高コスト・遅延）が唯一の手段で、少額取引はコストに合わない。
**After**: eBay経済圏が年間取引額240億ドル（2004年）に成長。PayPalはその決済手数料（2-3%）を取得。銀行はこの市場に参入できず、手数料収入を丸ごと逃した。

利潤が蒸発した側：小切手・郵便為替の手数料収入、銀行窓口の送金手数料。
利潤が集中した側：PayPal（決済手数料）、eBay（マーケットプレイス手数料）。

# 法則への接続

**LAW-009（ロックイン移動）**: インターネットが取引摩擦を崩壊させ、物理的支払インフラ（銀行支店網・ATM網・小切手処理システム）のロックインが溶解した。新しいロックインは「オンライン決済の両面ネットワーク」という新レイヤーに再形成された。銀行は旧ロックイン（支店網・規制ライセンス）に最適化されており、新レイヤーでは資産が負債に転化した（コンプライアンスの重さ、意思決定の遅さ、少額取引への無関心）。PayPalは新参者として旧資産を持たないため、新ロックインを直接構築できた。

**LAW-004（摩擦崩壊）**: 取引コストの崩壊がC2C電子商取引という新市場を生んだ（ジェボンズのパラドックス——取引が安くなると取引量が爆発する）。

**CON-004（因果証明コスト）**: PayPalのフィードバックループは即時（送金が成功するか失敗するか）。利用者は1回の取引で価値を体感する。信頼の前提条件も低い（最初は少額のeBay取引）。だからスタートアップが堀を作れた。銀行のオンライン決済サービスは同じ機能を提供したが、銀行口座の開設が前提（信頼の前提条件が高い）で、eBayコミュニティへの浸透戦略がなかった（両面ネットワークの片側しか持たなかった）。

# この事例が法則に反する側面（必須）

PayPalの成功は純粋なロックイン移動だけでは説明できない。eBayという既存プラットフォームの「寄生戦略」（eBayが公式決済を持たない空白に入り込んだ）が決定的だった。もしeBayが最初から自前決済を持っていたら、PayPalは入り込めなかった可能性が高い。つまりロックイン移動の窓は「特定の隣接プラットフォームが空白を持つ」という偶然的条件にも依存する——LAW-009の「構造的に窓が開く」という主張は、偶然要素を過小評価している可能性がある。

また、PayPalの初期成長はユーザー獲得ボーナス（新規登録で$10-$20）という赤字補填に依存しており、プロダクト優位性だけでネットワーク効果が自然発生したわけではない。ロックインの形成には資本の強制注入（LAW-003的な期待資本）が必要だった。
