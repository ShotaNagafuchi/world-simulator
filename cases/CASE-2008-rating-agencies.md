# 何が起きたか

格付け機関はもともと投資家（買い手）課金だった。1970年代、コピー機の普及で格付けレポートの複製が自由になり、買い手課金モデルが崩壊（評価結果は複製コストゼロの公共財になった）。各社は発行体（売り手）課金へ転換した。2000年代、仕組み金融（CDO）で発行体は最高格付けをくれる機関を選べる「レーティング・ショッピング」が常態化し、サブプライム関連 CDO の大半が AAA を得た。2007-08年に史上最大規模の一斉格下げが起き、金融危機調査委員会（FCIC）は格付け機関を危機の「key enablers」と名指しした。

# どの摩擦が下がったか

腐敗の起点は倫理の低下ではなく**摩擦低下**だった：複製コストの崩壊（コピー機）が買い手課金の回収可能性を壊し、収益源を評価される側へ押し出した（LAW-004 の意外な適用例——摩擦低下が中立性のコスト構造を壊した）。以後、審判の売上は「厳格さ」ではなく「格付けされる証券の量」に比例するようになった。

# 資金はどこからどこへ流れたか

発行体の手数料 → 格付け機関の仕組み金融部門（危機前の数年で収益が急拡大）。中立性の毀損コストは、格付けを信じた投資家と納税者に外部化された。対照群：買い手課金を守った Consumer Reports は中立を保ったが、公共財問題ゆえに規模はニッチに留まった。**中立性は保てるか儲かるかのどちらかで、両立の設計は難しい。**

# 法則への接続

- LAW-008: 主証拠。中立の審判は、評価される側から収益を得た瞬間に構造的に腐る。個人の倫理ではなく収益構造の問題であるため、参入者が善良でも再発する。
- LAW-005: 信用レイヤーの価値が上がるからこそ、その買収の価値も上がる。フィルタの価値と捕獲の誘因は同じ速度で成長する。
- LAW-004: 摩擦低下（複製コスト崩壊）が課金主体を反転させ、中立性を壊した因果連鎖。

# この事例が法則に反する側面（必須）

- 腐っても死ななかった。2008年後も issuer-pays は存続し、三大機関は規制（NRSRO 指定）を参入障壁とする寡占として高収益を維持している。**LAW-008 は「中立性が死ぬ」ことを予測するが「事業が死ぬ」ことは予測しない**——アウトプットの信頼と事業の存続は別物である。
- Google の organic/ads 分離のように、利益相反を制度設計（分離壁）で10年以上延命した例もある。腐敗は瞬間ではなく漸進であり、分離壁の設計品質が腐敗速度を決める。
