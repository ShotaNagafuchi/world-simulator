# 何が起きたか

1956年、McLean が改造タンカーで58個のコンテナを輸送。荷役コストは雑貨船の約 $5.86/トン から約 $0.16/トン へ、およそ1/36になった。港の滞在日数は週単位から時間単位になり、盗難・破損・保険料が急減した。規格化（ISO コンテナ）が1960年代に完了すると、世界の貿易構造が変わった。

# どの摩擦が下がったか

「船と陸の境界」という単一の摩擦。技術としては箱とクレーンにすぎず、発明的新規性はほぼない。**価値は発明ではなく、規格化と垂直統合（トラック—港—船の一貫システム）から生まれた。**

# 資金はどこからどこへ流れたか

港湾労働（雇用の大部分が消滅）と都心の旧港（NY マンハッタンの埠頭は壊滅、ニュージャージーへ移転）から、大型化に投資できた新港湾・船社・そして**最も遠くの安い労働力**へ。輸送費が距離の制約でなくなった瞬間、製造は東アジアへ再配置された。コンテナはグローバル・サプライチェーンという「新しい世界」の前提条件だった。

# 法則への接続

- LAW-004: 最強の証拠。摩擦低下（1/36）の直接の受益者は船社ではなく（海運業は恒常的な過当競争で低利益）、**摩擦低下を前提に事業を再設計できた第三者**（東アジア製造業、ウォルマート型小売、ジャストインタイム生産）だった。摩擦低下の価値は、下げた者ではなく、下がった世界を前提に組み直した者が取る。

# この事例が法則に反する側面（必須）

- McLean 自身は後年、燃料高騰への賭け（SL-7 高速船）で破産している。摩擦低下の実行者が価値を保持できる保証はない——LAW-004 は「誰が勝つか」を教えず、「どこで戦うべきでないか」しか教えない。この限界を予測に織り込むこと。
- 摩擦低下から資金流動の変化まで約15年（規格化完了まで）かかった。LAW-004 の時定数は技術の普及速度ではなく**規格・制度の合意速度**で決まる。
