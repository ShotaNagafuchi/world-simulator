# 何が起きたか

打ち上げコスト（$/kg）は1970年代のスペースシャトル以降、約40年間ほぼ横ばいで、業界全体が事実上の定数として扱っていた。SpaceX は2010年に Falcon 9（1回約6,000万ドル、シャトルの実効コスト約15億ドル/回の桁下）を投入し、2015年12月に第1段の着陸回収に成功、2017年に再使用飛行を実証した。LEO への輸送単価は数万ドル/kg 級から数千ドル/kg 級へ、約1桁下がった。

# どの摩擦が下がったか

物理は何も変わっていない。ロケット方程式も燃料も1970年代と同じである。マスクの操作は**制約の考古学**だった：

1. 物理層まで遡る——燃料費は打ち上げ価格の約1%以下。物理的下限と実勢価格の間に約100倍の乖離がある
2. 乖離の正体を特定する——cost-plus 契約（コストに利益率を上乗せする政府調達）はコスト削減の誘因を構造的に持たない。「定数」の正体は物理ではなく **institution 層のレント**だった
3. レントは物理と違って攻撃可能——垂直統合による部品内製（業界慣行の破壊）と機体再使用（「ロケットは使い捨て」という慣行の破壊）で価格を1桁下げた

# 資金はどこからどこへ流れたか

政府調達のレント（既存大手の随意契約的利益）から、固定価格契約の新規参入者へ。さらに打ち上げコストの1桁低下は、**それまで経済的に存在し得なかった需要**（数千機規模の通信コンステレーション＝Starlink、小型衛星産業）を出現させた。摩擦低下は既存市場の奪取ではなく市場の創造として回収された（ジェボンズ、LAW-004 の系）。

# 法則への接続

- LAW-007: 保守の誤り（速い層の制約を定数と誤認）の教科書。業界は institution 層のレントを physics 層の限界と誤分類していた。
- LAW-004: 摩擦（打ち上げ単価）の桁減が、宇宙産業のバリューチェーン全体（衛星設計思想・通信・地上局）を再配線した。
- LAW-003: 「火星移住」という検証不能だが追跡可能なナラティブが、実証前の資金調達を成立させた（CASE-2008-tesla と同型のマスタープラン装置）。

# この事例が法則に反する側面（必須）

- 「制約の考古学」だけでは足りなかった。乖離の存在は1980年代から複数の論者が指摘しており（知られていた）、実行に約100億ドル級の資本・10年超の期間・NASA COTS という**制度側の共犯**（固定価格契約への調達改革）が必要だった。定数の変数化は「見抜くこと」ではなく「見抜いたものを、制度が開く瞬間に、桁違いの資本で殴ること」である——タイミングと資本調達力が同格の必要条件。
- シャトルも「再使用でコストを下げる」という同じテーゼで作られ、**失敗した**（整備コストが使い捨てを上回った）。同じ正しい方向の賭けでも、実装の設計（何を再使用するか）で正反対の結果が出る。法則はポジションの方向を教えるが、実装の勝敗は教えない。
