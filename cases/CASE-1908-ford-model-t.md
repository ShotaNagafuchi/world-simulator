# 何が起きたか

1908年発売のモデルTは当初850ドル。1913年の移動組立ライン導入で組立時間が約12時間半から約1時間半へ短縮され、価格は1925年に260ドルまで低下した。累計1,500万台。自動車は富裕層の玩具から大衆インフラになった。

# どの摩擦が下がったか

製造（組立工数）という単一の摩擦が約1/8になった。設計思想は「製品の多様性を捨てて工程の反復性を買う」——黒一色はその帰結である。

# 資金はどこからどこへ流れたか

自動車製造の利潤は競争で急速に薄くなり（数百社が乱立→淘汰）、価値は**車の外側**へ移動した：ガソリン供給網（スタンダード・オイル系）、道路建設（公共投資の恒常化）、郊外不動産、モーテル・外食。車を作る摩擦が消えた結果、「車が走るための残った摩擦」＝燃料・道路・生活圏の再設計に価値が再集中した。

# 法則への接続

- LAW-004: 教科書的事例。摩擦低下は自動車産業を豊かにしたのではなく、バリューチェーンの利益プールを製造から燃料・インフラへ移した。予測すべきは「何ができるようになるか」ではなく「価値がどこへ逃げるか」だった。
- LAW-003: 1900年代の自動車スタートアップ乱立は期待資本の典型。数百社のうち生き残りは一桁%。

# この事例が法則に反する側面（必須）

- フォード自身は「摩擦を下げた当事者」でありながら価値を長期間保持した（LAW-004 は「摩擦を下げた工程から利潤が蒸発する」と言うが、蒸発には20年かかり、その間の独占利潤は莫大だった）。摩擦低下の**実行者**と**追随者**では時間軸が違う。
- GM は製品多様性（毎年のモデルチェンジ＝期待の再生産）でフォードを逆転した。摩擦最小化の一本槍は、期待の設計（LAW-003 の応用）に敗れ得る。
