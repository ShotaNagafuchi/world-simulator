# 制約の内容

人間は認知的一貫性を維持するために、自分の既存の信念・アイデンティティ・過去の選択と矛盾する情報を無意識に排除・歪曲する（Festinger 1957, Cognitive Dissonance; Munger の Inconsistency-Avoidance Tendency; Cialdini の Commitment & Consistency）。

帰結：
- 一度「Xは良い」と信じた人に「Xは悪い」というデータを見せても変わらない
- 一度投資した方向に追加投資する（サンクコスト効果）
- 「自分は○○派」というアイデンティティが、合理的な選択の乗り換えを阻害する

# 考古学（物理層までの遡行）

- **human-nature 層（不変）**: 認知的一貫性は社会的信用の基盤。「昨日と今日で違うことを言う人」は信用されない。一貫性の維持は社会的生存のための適応。しかし変化が必要な環境では足枷になる
- **市場への影響**: 投資家は損失ポジションを保持し続ける（自分の判断が間違っていたと認められない）。企業は失敗プロジェクトに追加投資する。消費者はブランドを乗り換えない（自分の過去の選択を否定することになるから）
- **LAW-003 との接続**: ナラティブに乗った投資家がナラティブの崩壊を認めない理由。「自分はAIの波に乗った賢い投資家」というアイデンティティが、損切りを妨げる
- **LAW-007 との接続**: 「AIが産業を変える」と信じた業界人が、ボトルネック不一致の証拠を見ても方向転換しない理由

# 予測への応用

- **市場バブルの持続期間**: バブルが弾けるまでの時間は、参加者のアイデンティティとの結びつきの強さに比例する。暗号資産が「コミュニティ」「文化」と結びついたとき、バブルの持続が延びる
- **技術採用の抵抗**: 既存技術のユーザーは新技術を「不要」と判断する傾向がある。これは合理的評価ではなく、一貫性維持の欲求
- **世界線の分岐予測（simulations/）**: 人間の判断が関与する分岐では、「証拠があっても信念を変えない期間」を織り込む必要がある

# 解凍条件の観測方法

- 認知的不協和の解消速度の変化。情報環境の変化（SNSでの速い合意形成）が一貫性バイアスを弱めるか強めるか
