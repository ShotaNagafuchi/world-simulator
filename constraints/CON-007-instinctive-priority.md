# 制約の内容

人間の経済行動の大半は、10の根源的欲求の代理満足に帰着する。「合理的であるべき」市場でも、実際に資金が流れる先は欲求の代理満足を提供する商品・サービスである。これは市場の歪みではなく定常状態。

## 10の根源的欲求（完全リスト）

| # | 欲求 | 定義 | 進化的根拠 |
|---|---|---|---|
| 1 | **生存** | 食べたい、健康でいたい、長生きしたい | 最も基本的な生物的命令 |
| 2 | **安全** | 脅威から守られたい、失いたくない | 資源保全、CON-010（損失回避）の根 |
| 3 | **つながり** | 相手と繋がりたい、愛されたい、親密でいたい | 社会的動物の生存条件 |
| 4 | **所属** | 集団に属したい、仲間外れにされたくない | 群れからの排除＝死の環境 |
| 5 | **地位** | 認められたい、他者より上に立ちたい | 序列上位＝資源へのアクセス |
| 6 | **自律** | 自分で決めたい、自由でいたい、支配されたくない | 行動の自由＝適応の柔軟性 |
| 7 | **互恵** | 恩を返したい、贈りたい、公正に扱われたい | 協力関係の維持（Cialdini Reciprocity） |
| 8 | **好奇心** | 知りたい、理解したい、探求したい | 環境の予測可能性の向上 |
| 9 | **快楽** | 気持ちよくなりたい、楽しみたい | 報酬系（ドーパミン）による行動強化 |
| 10 | **意味** | 意味のある人生を送りたい、遺したい | 長期的な集団貢献（利他行動の進化） |
| — | **性** | パートナーを得たい、魅力的でいたい | 生殖（生存・つながり・地位の複合） |

注：性は独立欲求というより生存・つながり・地位の交差点。市場分析上は独立カテゴリとして扱う。

# 考古学（物理層までの遡行）

- **human-nature 層（不変）**: 10の欲求は進化的に固定された動機系。大脳皮質による「合理的判断」は欲求の上に薄くかぶさった後付けの層であり、矛盾した場合は欲求が勝つ
- **欲求の階層**: 生存 > 安全 > つながり/所属 > 地位 > 自律/互恵 > 好奇心/快楽 > 意味。下位が脅かされると上位は後回しになる。ただし厳密な階層ではなく、状況依存で優先度が入れ替わる
- **支出のトレーサー分析**（欲求への帰着経路）:

| カテゴリ | 表の需要 | 根源的欲求 |
|---|---|---|
| ヘルスケア | 治療 | 生存 |
| 保険 | リスク管理 | 安全（損失回避） |
| SNS（時間支出） | つながり | 地位（いいね）＋所属＋快楽 |
| メッセージング | 連絡 | つながり |
| 高級ブランド | 品質 | 地位 |
| 教育（大学） | 知識 | 地位＋所属（好奇心ではない） |
| ギフト市場 | 感謝 | 互恵＋つながり |
| 検索/ChatGPT | 調べもの | 好奇心（＋安全：間違えたくない） |
| ゲーム/エンタメ | 楽しみ | 快楽＋地位（ランキング） |
| 起業/宗教 | 目標 | 意味＋地位 |

| 自動車 | 移動 | 自律（自由）＋地位 |
| 暗号資産投機 | 分散金融 | 快楽＋地位（一攫千金） |
| フィットネス | 健康 | 性（魅力）＋地位 |

# 予測への応用

**予測テンプレート**: 新しい技術/サービスが普及するかの判定：
1. どの根源的欲求を満たすか？（満たさないなら普及しない）
2. 既存手段と比べて欲求満足の効率は何倍か？（10倍なら CON-006 が発動）
3. 複数の欲求を同時に満たすか？（多いほど強い：SNS＝地位＋所属＋快楽で3重）
4. 欲求の階層でどこに位置するか？（生存・安全は市場が大きく安定、快楽は選択肢が多く競争激烈）

**欲求別の市場分析は `state/STATE-fundamental-needs.md` を参照。**

# 解凍条件の観測方法

- 消費支出の欲求帰着率の変化
- AIエージェントによる「感情を排除した購買」の割合
- 欲求に訴求しないサービスの市場シェア推移

# 依拠している法則・ポジション

LAW-003（期待資本）、LAW-009（ロックイン移動）の行動メカニズムの根底。CON-005（社会的証明）は「所属」の特殊ケース。CON-006（利便性＞信用）は「快楽・地位への近道」。CON-008（能力拡張＞判断代替）は「自律」の帰結。CON-009（インセンティブ超反応）は欲求が行動を駆動するメカニズムそのもの。
