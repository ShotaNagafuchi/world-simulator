# 制約の内容

人間の経済行動の大半は、5つの本能的欲求の代理満足に帰着する：**地位（他者より上に立つ）、性（パートナーの獲得・維持）、安全（脅威からの保護）、所属（集団への帰属）、快楽（即時の報酬）**。「合理的であるべき」市場（教育、金融、医療）でも、実際に資金が流れる先は本能の代理満足を提供する商品・サービスである。

これは市場の歪みではなく定常状態。本能に反するビジネスモデルは、たとえ合理的に優れていても採用されにくい。本能に沿うビジネスモデルは、たとえ合理的に劣っていても急速に普及する。

# 考古学（物理層までの遡行）

- **human-nature 層（不変）**: 5つの欲求は進化的に固定された動機系。大脳皮質による「合理的判断」は本能の上に薄くかぶさった後付けの層であり、本能と矛盾した場合は本能が勝つ（二重過程理論：System 1 >> System 2 in resource allocation）
- **支出のトレーサー分析**（本能への帰着経路）:

| カテゴリ | 表の需要 | 本能の代理満足 |
|---|---|---|
| 高級ブランド | 品質 | 地位シグナル |
| SNS（時間支出） | つながり | 地位の可視化（いいね＝地位通貨） |
| 教育（大学） | 知識 | 学歴＝地位シグナル＋所属シグナル |
| 暗号資産投機 | 分散金融 | 一攫千金（快楽）＋先行者（地位） |
| フィットネス | 健康 | 性的魅力＋地位シグナル |
| 保険 | リスク管理 | 安全（不安の除去） |
| 寄付・慈善 | 利他 | 地位シグナル＋所属シグナル |
| AI生成拒否 | 品質管理 | 「人間が作った」＝地位シグナル |

- **結果**: 新しい技術が「合理的に優れている」ことは普及の必要条件ではない。その技術が本能的欲求のどれかをより効率的に満たすことが十分条件。

# 予測への応用

**LAW-003（期待資本）との結合**: ナラティブが売れるのは人間が合理的だからではなく、「乗り遅れたくない」（所属）と「先行者になりたい」（地位）が同時に発火するから。ナラティブの強度は、活性化する本能的欲求の数と強度で予測できる。

**LAW-005（信用レイヤーの価値）との結合**: 信用レイヤーが価値を持つのは「合理的に情報を選別したい」からではなく、「間違った選択をして地位を失いたくない」（安全＋地位）から。信用の需要は合理性ではなく不安（安全欲求）に根ざす。

**LAW-009（ロックイン移動）との結合**: 新参者が旧独占者を倒す「10倍便利」（CON-006）は、合理的便利さではなく本能的満足の便利さ。TikTokがYouTubeから注意を奪ったのは「より良いコンテンツ」ではなく「より速い快楽フィードバックループ」。Instagramが写真共有から勝ったのは「より良い写真」ではなく「より効率的な地位シグナリング」。

**予測テンプレート**: 新しい技術/サービスが普及するかの判定：
1. どの本能的欲求を満たすか？（満たさないなら普及しない）
2. 既存手段と比べて本能的満足の効率は何倍か？（10倍なら CON-006 が発動）
3. 複数の欲求を同時に満たすか？（多いほど強い：SNS＝地位＋所属＋快楽で3重）

# 解凍条件の観測方法

- 消費支出の本能帰着率の変化（難しいが、広告のアピール軸の分析で代理観測可能：地位・安全・快楽訴求の広告が減少するか）
- AIエージェントによる「感情を排除した購買」の割合
- 本能的訴求を排除したサービスの市場シェア推移

# 依拠している法則・ポジション

LAW-003（期待資本）、LAW-009（ロックイン移動）の行動メカニズムの根底にある。CON-005（社会的証明）は本制約の「所属欲求」の特殊ケース。CON-006（利便性が信用を超える）は「快楽・地位への近道が開いた」ケース。

この制約が解凍したら（人間が本能に従わなくなったら）、ビジネスモデルの予測枠組み全体が崩壊する。ただしhuman-nature層であるため、技術的な時間軸（年-10年）では動かない。
