# 制約の内容

人間は、現在の状態（デフォルト）を「正常」と認識し、そこからの変化に不均衡なコストを感じる。臓器提供のオプトイン/オプトアウトで参加率が10倍変わる（Johnson & Goldstein 2003）ように、デフォルトの設計がシステムの挙動を決定する。

これは怠惰ではない。変化には3つの構造的コストがある：
1. **認知コスト**: 新しい選択肢を評価する思考エネルギー
2. **損失コスト**: 現状を失うリスク（CON-010 で2倍に増幅）
3. **アイデンティティコスト**: 「自分は○○を使う人」という自己認識の書き換え（CON-012）

3つの合計が、新しい選択肢の利得を上回る限り、人は動かない。

# 考古学（物理層までの遡行）

- **human-nature 層（不変）**: 認知資源の節約。脳のエネルギー消費は体重の2%の質量で20%のエネルギーを使う。新しい判断はコストが高い。デフォルト維持はエネルギー効率の最適化
- **CON-006 の裏面**: 「10倍便利でないと動かない」の理由がこれ。2-3倍では3つのコストの合計を超えない。10倍になって初めて超える
- **LAW-007 との接続**: ボトルネックでない摩擦を桁で下げても産業が変わらない理由の行動的説明。利用者にとって「変わるべき理由」がボトルネック解消でなければ、3つのコストを超える利得が生まれず、現状維持が選ばれる

# 予測への応用

- **普及予測**: 新技術の普及速度 = f（利得 − [認知コスト + 損失コスト×2 + アイデンティティコスト]）。この値が正でないと移行が始まらない
- **デフォルト設計**: 自分がプロダクトを作る側なら、ユーザーのデフォルトに自分を埋め込むことが最も強い戦略。変更コストを課す必要すらない——デフォルトの位置にいるだけで維持される
- **LAW-009 との結合**: 旧ロックインが溶解しても、利用者が現状維持バイアスで旧サービスに留まる期間がある。これが「窓の時間幅」を決める。窓は「旧サービスの劣化速度 vs 現状維持バイアスの強さ」で決まる

# 解凍条件の観測方法

- デフォルト効果（オプトイン/オプトアウトの差）の経年変化
- AI時代に「AIが最適を選ぶ」が新しいデフォルトになるか

# 依拠している法則・ポジション

CON-006（10倍閾値）の説明原理。CON-010（損失回避）の行動的帰結。LAW-007の行動メカニズム。LAW-009の窓の時間幅予測。
