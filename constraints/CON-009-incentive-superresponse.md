# 制約の内容

人間の行動を予測するとき、信念・価値観・能力ではなくインセンティブ構造を見ろ。報酬がある方向に人は動き、罰がある方向から人は逃げる。これは例外なく適用される。善人も悪人も、専門家も素人も、インセンティブの関数として行動する。Munger: "Show me the incentive and I'll show you the outcome."

システム設計の帰結：インセンティブを変えずに行動を変えようとする試みは必ず失敗する。教育・啓蒙・道徳的説得は、インセンティブ構造が変わらない限り持続的な行動変化を生まない。

# 考古学（物理層までの遡行）

- **human-nature 層（不変）**: 報酬追求と罰回避は神経回路レベルで実装されている（ドーパミン系）。意識的な判断よりも速く、強く作動する。「インセンティブに反して行動する」ことは、「重力に逆らって歩く」に等しい認知的コスト
- **LAW-002 の裏の原理**: 貴重な発信源が発信しない理由 = 発信のインセンティブがない（信用は飽和、リスクは残る）。これはインセンティブ超反応の直接的帰結
- **LAW-008 の裏の原理**: 審判が腐る理由 = 腐る方向にインセンティブがある（甘い評価 → 顧客が増える）。個人の倫理と無関係に、インセンティブが結果を決める
- **LAW-003 の裏の原理**: 期待に資金が流れる理由 = 「期待を売る」側にインセンティブがある（実成果より期待の方が売りやすい）

# 予測への応用

あるシステムの挙動を予測するとき：
1. 参加者のインセンティブ構造を列挙する（報酬は何か、罰は何か、誰が払うか）
2. インセンティブの方向が望ましい結果と一致しているか確認する
3. 不一致がある場合、**参加者は必ずインセンティブに従い、望ましい結果は実現しない**
4. 結果を変えたければインセンティブを変えろ。行動を変えようとするな

# 解凍条件の観測方法

- インセンティブと行動の不一致が主流経済で持続する事例の蓄積（宗教的動機や思想的動機による経済的に非合理な行動が、市場規模で影響を持つか）

# 依拠している法則・ポジション

LAW-002、LAW-003、LAW-008 の共通の裏の原理。これら3法則はすべてインセンティブ超反応の特殊ケースとして導出できる。この制約が解凍したら、3法則のメカニズムが崩壊する。
