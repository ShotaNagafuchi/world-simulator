# 制約の内容

「プロ品質の文章・編集・翻訳には専門人材の時間が必要」という制約。2022年まではコンテンツ産業全体がこれを事実上の定数として価格設定していた（ライター単価、翻訳単価、制作費）。

# 考古学（物理層までの遡行）

- 正体は physics ではなく tech-cost 層だった：文章生成に物理的下限はなく、「人間にしかできない」は当時の技術水準の話にすぎなかった
- 定数だと信じられていた期間が長かったため、解凍の瞬間に産業全体の価格体系が崩れた（ライティング・翻訳の受託単価の急落）
- **教訓：層の誤認の実例。** tech-cost 層の制約を human-nature 層（「創造性は人間だけ」）と誤分類していた者ほど、解凍で被弾した

# 解凍条件の観測方法

解凍済み。このファイルは「定数が変数になる」ライフサイクルの参照例として保持する。解凍後の帰結は LAW-004 → LAW-005 の通り：生成の利潤は蒸発し、価値は残った摩擦（CON-002 責任、CON-001 注意/引用枠）へ再集中しつつある。

# 依拠している法則・ポジション

この解凍こそが WORLD-2026-001 の初期条件である。次に「どの定数が同じ道を辿るか」を探すことが、constraints/ ディレクトリの存在意義。候補：発信誘因（LAW-002 の対価設計による解凍）、検証コスト（出所証明の標準化による解凍）。
