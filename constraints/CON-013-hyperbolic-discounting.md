# 制約の内容

人間は「今日の$100」を「1年後の$120」より強く選好する。この割引は指数的（合理的）ではなく双曲的（不均衡）——近い将来の差は大きく感じ、遠い将来の差は小さく感じる。結果：

- 今すぐの報酬がある選択肢が、長期的に最適な選択肢に勝つ
- サブスクリプションの解約は「今月の損」を避けるために先延ばしされる
- 長期投資（研究開発、ブランド構築、インフラ）は常に短期施策（広告、キャンペーン）に予算を奪われる

# 考古学（物理層までの遡行）

- **human-nature 層（不変）**: 進化的に、不確実な環境では「今確実に得られる報酬」を優先することが適応的。遠い将来は到来しないかもしれない。この選好は大脳辺縁系に実装されており、前頭前皮質による「合理的な長期計画」は常にこの力に抗う
- **LAW-003 との接続**: 期待資本が成果資本に勝つ理由の一端。「今の期待」は即座に資金を動かすが、「将来の成果」は割り引かれて評価される。ナラティブは今ここにあり、成果は未来にしかない
- **LAW-006 との接続**: ゴールドラッシュの道具屋が儲かる理由——道具の報酬は「今すぐの売上」、金の報酬は「将来の不確実な産出」。双曲割引により、参加者は道具への支出（即時の行動感）を金の産出（遠い報酬）より重視する

# 予測への応用

- **プロダクト設計**: 即時の報酬ループ（即座の結果表示、すぐ使える、今日から効果がある）を持つプロダクトは、長期的に優れた結果を出すが即時報酬のないプロダクトに勝つ
- **CON-004 との結合**: 因果証明コストが高い＝報酬が遠い。因果証明コストが低い＝報酬が近い。CON-004のフィードバックループの速さが重要な理由は、双曲割引により「遅い報酬は報酬として認識されない」から
- **市場タイミング**: 「正しいが早すぎる」技術は失敗する。利用者にとって報酬が遠すぎるから。同じ技術でも、即時の報酬を提供するパッケージングに変えると普及する

# 解凍条件の観測方法

- 長期投資比率の構造的変化（企業のR&D比率、個人の貯蓄率）
- 「今すぐの結果」を求める傾向の強化/弱化

# 依拠している法則・ポジション

LAW-003（期待が成果に勝つ理由）、LAW-006（道具屋が儲かる理由）、CON-004（フィードバック速度が重要な理由）の共通基盤。
