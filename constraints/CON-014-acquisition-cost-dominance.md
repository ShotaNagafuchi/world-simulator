# 制約の内容

人間の意思決定を支配するのは情報の品質ではなく、その情報の取得コスト（時間・認知負荷・労力）である。取得コストフィルターは品質評価の**前に**作動する。閾値を超えない情報は品質を評価される機会すらない。閾値を超えた情報は、流暢さ・利用可能性ヒューリスティクスにより「正しい」と知覚される。

結果：**最初に届いた情報が意思決定を支配する。** 2番目以降の情報は、追加取得コスト > 限界利得のため、取得すらされない。

# 先行研究（8つの研究伝統の統合）

この原理は複数の研究伝統で独立に発見されているが、統一された名前がない：

1. **Satisficing**（Simon 1955）: 人間は最適化しない。最初の「合格」解で検索を止める
2. **情報の経済学**（Stigler 1961）: 情報取得にもコストがあり、限界コスト > 限界利得で検索終了
3. **Diamond Paradox**（Diamond 1971）: わずかな検索コストでも、均衡では検索がゼロに崩壊する。小さな取得コストが巨大な下流効果を持つ
4. **Availability Heuristic**（Tversky & Kahneman 1973）: 思い出しやすい（＝取得コストの低い）情報を重要・高頻度と判断する
5. **WYSIATI**（Kahneman 2011）: 今見えている情報から最も整合的なストーリーを構築し、見えていない情報の存在に気づかない。不足を感じない
6. **Principle of Least Effort**（Zipf 1949）: あらゆる情報行動で人間は最小努力を選ぶ。専門家でも例外ではない
7. **Information Foraging Theory**（Pirolli & Card 1999）: 情報の質ではなく「質/コスト比」で行動が決まる。取得コストがゼロに近い情報は、低品質でも高「質/コスト比」になる
8. **Fluency Heuristic**（Reber et al.）: 認知的に処理しやすい情報を「正しい」「信頼できる」と感じる。取得コストが低い情報は処理も流暢 → 信頼も高い

# 考古学（物理層までの遡行）

- **human-nature 層（不変）**: 認知資源の制約（脳は体重の2%で20%のエネルギーを消費）。情報評価は高コストな認知活動であり、脳はフィルタリングで負荷を最小化する。取得コストフィルターは意識的判断ではなく、神経回路レベルで実装されたゲート機構
- **CON-013（双曲割引）との接続**: 情報の品質改善は「遠い報酬」（良い意思決定は将来判明する）。情報の取得容易さは「今の報酬」（今すぐ楽に入手できる）。双曲割引により「今の楽さ」が「将来の品質」に構造的に勝つ
- **CON-011（現状維持バイアス）との接続**: 追加検索 = 「現状（今の知識）からの変化」。変化コストを過大評価 → 追加検索しない

# 帰結（予測への応用）

## 1. 保険の営業マンが比較サイトに勝つ理由

```
営業マンの情報: 取得コスト = 0（向こうから来る）
  → 閾値超え → 品質評価される → 流暢に話す → 「正しそう」→ 契約
比較サイトの情報: 取得コスト = 中（自分でアクセスし、読み、比較する）
  → 閾値を超えない → 評価すらされない
```

不動産、金融商品、B2Bサービスも同じ構造。「来てくれる人」の情報が勝つ。

## 2. AI検索の第一回答が独占する理由

```
AI第一回答: 取得コスト ≈ 0（聞けば即座に出る）
  → WYSIATI: 他の答えの存在に気づかない
  → Fluency: 流暢に答える → 「正しい」と知覚される
  → Diamond Paradox: 追加検索の期待利得 ≈ 0 → 検索終了
AI第二回答以降: 取得コスト > 0（追加質問、スクロール、別サービスに切替）
  → 取得すらされない確率が高い
```

したがって**AIの引用枠の1番目に出ることの価値は、2番目以降と非連続的に異なる**。これは検索の1ページ目と2ページ目の差（連続的）とは構造が違う。

## 3. LAW-001（情報の価値 = 希少性 × 変換率）への接続

情報の「変換率」を構成する4つの工程（発信→構造化→信用付与→意思決定接続）のうち、本制約は**意思決定接続**の工程を支配する。情報がいくら高品質でも、意思決定者に「取得コスト低く」届かなければ、意思決定に接続されない。変換率のボトルネックは品質ではなく到達の容易さ。

## 4. LAW-009（ロックイン移動）への接続

ルーティングの価値が高い理由の行動的基盤。ルーター（Google、AI）は情報の取得コストをゼロにする。取得コストがゼロの情報が意思決定を支配するから、ルーターを経由した情報が世界を支配する。

# 解凍条件の観測方法

- AI検索の利用者が第一回答を超えて追加検索する率の変化
- 情報品質を独立評価するツール/習慣の普及率
- 「AIに聞いた → そのまま実行した」の割合と、「AIに聞いた → 別ソースで確認した」の割合

# 依拠している法則・ポジション

LAW-001（変換率のボトルネック）、LAW-002（発信しない理由＝取得コストが高い情報は意思決定に到達しない）、LAW-005（信用レイヤーの価値上昇——ただし信用レイヤーも取得コストフィルターに支配される）、LAW-009（ルーティングの価値の行動的根拠）。

この制約が解凍したら、「AIの第一回答が支配する」という構造が崩れ、情報の質による競争が復活する。ただしhuman-nature層であるため、解凍は技術的時間軸では起きない。
