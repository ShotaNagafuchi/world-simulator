# 制約の内容

人間は選択の不確実性が高いとき、「他の多くの人が選んでいるもの」をデフォルトで選ぶ。これは認知コストの節約（他者の選択を情報として利用する）であり、合理的な近似である。結果として、初期の利用者数の差が自己強化ループを形成し、「多い方がさらに多くなる」力学が働く。

これはロックインの**人間側の基盤**である。ルーティング機能（LAW-009）がロックインの構造条件を作り、社会的証明（本制約）がロックインの行動条件を作る。両方が揃って初めてロックインが成立する。

# 考古学（物理層までの遡行）

- **human-nature 層（不変）**: 社会的証明は進化的に獲得されたヒューリスティック。不確実な環境で他者の行動を情報源として利用することは、個体の生存確率を高める適応行動。全文化・全時代で確認される（Cialdini 1984, Henrich 2015）。脳の社会的推論回路（mPFC, TPJ）に実装されており、訓練や意志で無効化できない
- **institution 層**: 社会的証明の制度化 = ランキング、ベストセラーリスト、「いいね」数の表示、「N人が購入」表示。制度は human-nature を増幅する装置
- **tech-cost 層**: ソーシャルメディアとレコメンドアルゴリズムが社会的証明のフィードバックループを高速化した（かつては口コミに数年かかった伝播が秒単位に）

**ロックインとの接続**: ルーティング（LAW-009）は構造的に「ここを通るしかない」を作る。社会的証明は行動的に「みんなが使っているからここを使う」を作る。Google のロックインは、ルーティング独占（構造）+ 「検索＝Google」という社会的常識（行動）の両方で成立していた。

**LAW-003（期待資本）との接続**: 「みんなが投資しているから投資する」は社会的証明の金融版。FOMO はこの制約の感情的表現。

# 解凍条件の観測方法

- AIエージェント経由の購買・意思決定の割合（人間が直接選ばない比率）
- AIエージェントの選択基準が社会的証明（人気度・レビュー数）に依存する度合い。AIが人気度を無視して客観指標で選ぶなら、この制約は迂回される

# 依拠している法則・ポジション

LAW-009（ロックイン移動）の「なぜルーティングにロックインが形成されるか」の行動基盤。LAW-003（期待資本）の根底にある人間行動原理。この制約が解凍したら、ロックインの形成力学が根本的に変わる——「みんなが使っている」が意味を持たない世界では、ネットワーク効果の自己強化ループが弱まり、独占が不安定化する。
