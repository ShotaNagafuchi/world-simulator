# 方法論

「そういうものだから」と受け入れている動作を網羅的にリストアップし、
AI/ChatGPTで**既に解決されたもの**と**まだ解決されていないもの**に分類する。
未解決のもので、全く別の方法で10倍良くなる可能性があるものが機会。
