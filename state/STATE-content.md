# 現在地サマリ

AI 検索（AI Overviews / ChatGPT / Perplexity / Gemini）経由の情報接触が主要チャネル化し、ゼロクリック化で出典サイトへの流入は構造的に減少中（SIG-2026-001）。AI 企業と出版社のライセンス契約は常態化フェーズに入った（PRED-2026-000 で true 解決、累計10件超）。AEO/AI 可視性計測のつるはし市場が形成期にあり、独立系（Profound 等）と既存 SEO スイート（Semrush / Ahrefs の AI 機能）が混在。個人専門家への対価プログラムはまだ未観測。出所証明（C2PA）は規格団体・カメラメーカー・一部プラットフォームの実験段階で、標準機能化には未到達。
