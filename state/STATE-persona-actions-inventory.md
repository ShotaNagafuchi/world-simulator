# 方法論

特定のペルソナ（職業/趣味）ごとに「そういうものだと思っている動作」をリストし、
12フィルターで機会を探す。一般消費者版（STATE-normal-actions-inventory）の拡張。

重要な前提:
- F1（必須行動）はペルソナ内での頻度で判定する（全人口ではなくそのペルソナの人口×頻度で市場規模を算出）
- ペルソナ人口が少なすぎる場合（日本1万人以下等）は市場として成立しない
- 複数ペルソナに共通する動作は一般版に統合すべきシグナル
