# 記録

CASE-2008-tesla と LAW-004（摩擦低下）の原料。「EV 普及は思想ではなくこのコスト曲線の関数」の根拠となる数値系列。

| 年 | 価格（$/kWh、パックベース概算） |
|----|-------------------------------|
| 2010 | ~1,100 |
| 2013 | ~650 |
| 2016 | ~300 |
| 2019 | ~156 |
| 2021 | ~132 |
| 2023 | ~139（原材料高で一時反転） |

観察（解釈は laws/ の仕事だが、系列そのものの特徴として）：

- 2010-2021 でおよそ 1/8。年率約 18% の低下（学習曲線：累積生産量が倍になるごとに一定率で低下）
- 2022-23 に原材料（リチウム・ニッケル）価格で初の反転。学習曲線は原料制約で折れ得ることの記録
- ガソリン車と購入価格で競合する目安とされた $100/kWh は、単純外挿より遅れて到達しつつある

更新時は BloombergNEF の年次サーベイを追記する（追記のみ、過去の値は書き換えない）。
